Для запуска приложения нужно удовлетворить списку требований:
1. Должен быть запущен mysql на порту 3306.
2. Должна быть создана БД finra
3. Приложение подключается к БД под учётной записью 
login: root 
password: mysql
Эти настройки можно отредактировать в файле pom.xml в разделе "profile".
При старте приложения оно будет пересоздавать структуру таблиц в БД "finra".

Для запуска приложения используйте команду:
mvn clean package spring-boot:run

Для проверки работы приложения была создана страница с элементарной формой позволяющей загрузить файл.
Приложение стартует на порту 80(и 443, поэтому если один из этих портов занят - приложение выдаст соответствующую ошибку),
 в корневом контексте, поэтому для того что бы попасть на эту страницу достаточно вбить в браузере:
http://localhost
на странице с формой можно указать имя файла (которое будет использовано при сохранении файла в файловую систему) под которым он будет храниться в БД и значение свойства (some property), так как в задании было указано что должно быть несколько "методанных", которые должны сохраняться в БД.
После загрузки файла на сервер - в ответ - приложение пошлёт json с id сохранённого файла, который отобразится на странице к примеру:
{"id":1}
Для получения информации о файле по его id надо использовать урл вида:
http://localhost/files/?id=1
в ответ на такой запрос приложение вернёт json объект вида:
{"id":1,"name":"123.jpg","someProperty":"zz","creationTime":"2017-09-04 05:14:29.0","size":6566}
Для загрузки файла из приложения к себе - надо использовать урл вида:
http://localhost/files/download/?id=1
